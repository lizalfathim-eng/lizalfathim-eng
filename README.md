## 🚀 Projects

[**AetherAI**](https://github.com/lizalfathim-eng/AetherAi)
*AI-Powered Document Search System*
A full-stack Django web application built for Kerala Finance Department staff to retrieve Government Orders and circulars using natural language queries. Features OCR-based PDF ingestion, paragraph-level chunking, and a hybrid-ranked semantic search engine using Sentence Transformers (all-mpnet-base-v2) and FAISS vector indexing.

[**ChromaNav**](https://github.com/lizalfathim-eng/Chromanav)
*Colour-Blind Traffic Signal Detection*
A real-time desktop application using YOLOv8 and OpenCV that helps colour-blind users identify traffic signals and road signs. Combines HSV colour-space analysis, confidence-threshold filtering, and text-to-speech voice feedback for a fully non-visual accessibility layer.
